# 個人情報マスキングツール fuseji.jp《ふせじ》

ブラウザ上で完結し、外部へのデータ送信を一切行わない安全な個人情報マスキングツールです。

[![Version](https://img.shields.io/badge/version-2.61-blue)](https://fuseji.jp)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

🌐 **オンライン版**: [fuseji.jp](https://fuseji.jp/)

---

## 特徴

- **完全ローカル処理** — テキストは一切外部に送信されません
- **インストール不要** — ブラウザ（またはHTMLファイルをダブルクリック）で即使用可能
- **対話的なマスキング** — 自動判定 + 手動修正で精度を高められます
- **5つのマスク形式** — `{姓名1}` 形式のほか、英語ラベル・可逆伏字・非可逆伏字・疑似データ置換を選択可能
- **逆変換機能** — AIに処理させた結果を元のテキストに復元できます
- **辞書機能** — よく使う人名・法人名・語句をブラウザに永続保存（保存は同意制）

---

## 使い方

1. HTMLファイルをブラウザで開く（またはダブルクリック）
2. 左側のテキストエリアに個人情報を含むテキストを貼り付ける
3. **⚡ 自動判定** を押す — メール・電話・住所・氏名・法人名などを一括検出
4. チップを確認し、不要なものは ✕ で削除
5. **マスキング実行**（または Ctrl+Enter）で右側にマスキング済みテキストを生成
6. 右側テキストを生成AIに貼り付け、AIの出力を「逆変換」パネルで元テキストに復元

> **入力サイズの上限**: 自動判定・マスキング処理は20万字までを対象とします。これを超えるテキストはエラー表示となるため、分割してご利用ください。

## マスク形式と疑似データ置換（v2.61 新機能）

⋯ メニューから5つの出力形式を択一で選べます（選択中の項目に ✓ が付きます）。

| 形式 | 出力例 | 逆変換 | 用途 |
|---|---|---|---|
| `{ラベル}`（既定） | `{姓名1}` `{株式会社1}` | ○ | 日本語プロンプト向け |
| `[LABEL]` | `[NAME_1]` `[ORG_1]` `[EMAIL_1]` | ○ | 英語プロンプト向け |
| `●■◆`（伏字・可逆） | `●■◆` `●▓◆`（項目ごとに異なる記号列） | ○ | 伏字の見た目のまま逆変換したい場合 |
| `●●`（伏字・非可逆） | すべて `●●` | ✕ | 公開・共有用の墨消し |
| 疑似データ置換 | 高橋健一 → 日向和奏 | ○ | 自然な文章のまま伏せ字化 |

### 疑似データ置換

`{ラベル}` の代わりに、**元テキストに含まれない架空の値**へ置き換えます。一見すると自然な文章ですが、氏名・法人名・住所・電話番号などがすべて書き換わっています。

- 同一人物のフルネーム・姓・名は一貫した架空人物になります（原文のスペース有無も再現）
- 法人は種別を維持（学校法人→○○学院など。株式会社の前置/後置も原文どおり）
- 電話・各種番号は桁数と区切りを保ってランダム化。メール・URLはRFC予約の example ドメインを使用
- 対応リスト・逆変換は通常どおり使えます

> **注意**: 架空値は無作為に生成するため、実在の名称・番号と偶然一致する可能性があります。

### 可逆伏字（●■◆）

先頭 `●` + 記号列 + 終端 `◆` という構造のラベルを項目ごとに割り当てます。この構造により、ラベル同士が隣接したり本文の記号と連続しても、逆変換が混線しません。本文に既に含まれる記号の並びは避けて採番します。

### 補足

- 非可逆の `●●` 形式では対応リストを作りません（全項目が同一文字列になるため）。逆変換もブロックされます
- 手動で追加したカスタム伏字のラベルは、どの形式でもユーザー指定のまま維持されます
- 本文に `{姓名1}` などラベルと同じ文字列が既にある場合、その番号をスキップして採番するため、逆変換で本文側が書き換わることはありません

### マスク形式の記憶

ブラウザ保存に同意済みの場合のみ、選択したマスク形式を記憶し、次回起動時に復元します（復元時はステータスに明示します）。未同意の場合はセッション限定で、ページを閉じると既定の `{ラベル}` に戻ります。「保存データを全削除」で記憶も消え、既定に戻ります。

## ブラウザ保存と同意（ver.2.59〜）

辞書・ブロックリスト・マスク形式などのブラウザ保存はすべて**同意制**です。同意するまでブラウザには何も保存されません（セッション中はメモリ上で動作します）。⋯ メニュー →「⚙ 辞書・ブロックリスト管理」から、同意の変更と「保存データを全削除」ができます。

---

## そのほかの主な機能

- **対応リストの書き出し / 読み込み** — ラベルと原文の対応をJSONまたはTSVで書き出し、別セッションで読み込んで逆変換できます
- **語句候補スキャン**（ver.2.58〜） — 本文から伏せ字候補になりそうな語句を抽出して一覧表示します
- **手動伏字** — 右側テキストで範囲選択 → 右クリックで任意の語句を伏せ字にできます

## 外部辞書（オプション）

`address-dict.json`（地名辞書）と `corp-dict.json`（法人名辞書）を読み込むと、自動判定の精度が向上します。辞書データはブラウザのIndexedDBに保存されます（外部送信なし）。

読み込み時には語句の長さ（40字以内）・文字種（漢字・かな・カタカナ・英数等）・制御文字の有無をチェックし、不正なエントリを除外します。

### オンライン版（fuseji.jp）での使い方

1. ⋯ メニュー →「⚙ 辞書・ブロックリスト管理」を開く
2. **外部辞書** セクションの「⬇ 取得」ボタンを押す
3. fuseji.jp から辞書を自動取得してIndexedDBに保存されます

> **注意**: 「⬇ 取得」ボタンを押したときだけ fuseji.jp に通信します。テキストのマスキング処理自体は常に完全ローカルです。

### ローカルファイルでの使い方

ローカルファイル（`file://`）で開いた場合、セキュリティ上の制約によりネットワーク取得はできません。「⬇ 取得」ボタンは表示されません。

1. [fuseji.jp/address-dict.json](https://fuseji.jp/address-dict.json) と [fuseji.jp/corp-dict.json](https://fuseji.jp/corp-dict.json) をあらかじめダウンロードしておく
2. ⋯ メニュー →「⚙ 辞書・ブロックリスト管理」を開く
3. **外部辞書** セクションの「📂 読み込む」で保存したJSONファイルを選択

### オンライン版とローカル版の違い

| 機能 | fuseji.jp（オンライン） | ローカルファイル |
|---|---|---|
| テキストのマスキング処理 | 完全ローカル | 完全ローカル |
| 辞書の「⬇ 取得」ボタン | 使用可能 | 非表示 |
| 辞書の「📂 読み込む」 | 使用可能 | 使用可能 |
| 辞書・設定の保存先 | fuseji.jp のブラウザ領域 | file:// のブラウザ領域 |

辞書・設定（マスク形式の記憶を含む）の保存先はオリジン（開き方）ごとに独立しています。オンライン版で登録した辞書はローカル版では引き継がれません。

## v2.61 の主な変更点

### 新機能

- 疑似データ置換モードを追加（架空の氏名・法人名・住所・番号などへの置き換え）
- マスク形式の選択を追加（`{ラベル}` / `[LABEL]` / `●■◆` 可逆伏字 / `●●` 非可逆伏字 / 疑似データ置換の択一）
- マスク形式の記憶と復元（ブラウザ保存に同意済みの場合のみ）

### 修正

- 本文にラベルと同じ文字列（`{姓名1}` など）が既に含まれる場合、逆変換で本文側まで書き換わる問題を修正（該当番号をスキップして採番）
- 電話番号の区切り一致を厳格化し、スペース区切りの無関係な数字列への過剰マッチを防止
- 電話・メール・URLの表記違い（区切り・大文字小文字）を出現表記ごとに別ラベル化し、逆変換で元の表記どおり復元されるよう修正
- 手動登録人物から派生する1文字の姓・名の単独置換を廃止（「翔→雄翔館」のような語中の誤置換を防止。フルネームと姓のみ登録は従来どおり）
- ローマ字名の二重処理を解消し、専用処理に一本化
- 氏名の逆変換原文を「本文で最初にマッチした表記」に変更（登録形とのスペース差で逆変換結果が変わる問題を修正）

## 辞書ファイルの出典・ライセンス

### address-dict.json（地名辞書）

- **出典**: [Geolonia 住所データ](https://github.com/geolonia/japanese-addresses)（Geolonia株式会社）
- **元データ**: 国土交通省位置参照情報ダウンロードサービス、日本郵便 郵便番号データ
- **ライセンス**: [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/deed.ja)
- **加工内容**: 都道府県・市区町村名をフラットなリスト形式に変換し、fuseji辞書フォーマットに整形

> Geolonia 住所データ（CC BY 4.0）https://github.com/geolonia/japanese-addresses を加工して作成

### corp-dict.json（法人名辞書）

- **出典**: [国税庁法人番号公表サイト](https://www.houjin-bangou.nta.go.jp/)（国税庁）、EDINETコードリスト（金融庁）
- **ライセンス**: [公共データ利用規約（第1.0版）](https://www.houjin-bangou.nta.go.jp/riyokiyaku/)
- **加工内容**: 内国法人・組合のみを抽出し、全角文字の正規化・個人名の除外を行い、fuseji辞書フォーマットに整形

> 国税庁法人番号公表サイト（国税庁）https://www.houjin-bangou.nta.go.jp/ を加工して作成  
> このサービスの内容は国税庁によって保証されたものではありません
